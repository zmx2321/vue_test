# 此框架用来写一些临时案例

> https://zmx2321.github.io/vue_test

# 项目介绍

1. vue_test 是一个基于 vue2.0 的框架，用来写一些临时案例
2. vue2.0 + vue-router + vuex + axios + element-ui
3. 项目中用到的技术栈
   - vue2.0
   - vue-router
   - vuex
   - axios
   - element-ui
   - vue-cli
   - webpack
   - es6
   - less
   - eslint
   - babel
   - git
   - npm
   - yarn
   - vue-devtools
